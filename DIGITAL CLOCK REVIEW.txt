HTML:

Criar uma estruta de HTML básico com "HTML:5"
mudar o atributo de linguagem para "pt-br"
criar uma tag <link> Ela já vai gerar uma linha de código para você criar referência ao CSS:
<link rel="stylesheet" href="style.css">

nessa sessão do código ficará as fontes que você buscou no Google

dentro do Body é necessário criar algumas divs classificadas dentro do Body:

    <div class="relogio">
        <div class="display">00:00:00</div>
        <footer>Criado por Fábio Teixeira</footer>
    </div>
 
após isso basta criar uma TAG <script src="nome_do_arquivo.js"></script>
Fechar o Body

CSS:

Essa é a configuração de exibição das coisas na tela:

html, body {
    margin:;
    padding: ;
    background-color: ;
}


Depois basta mencionar as divs classificadas usando .


.relogio { /*Para puxar uma TAG referenciada no HTML, deve-se usar um . */
    border: 2px solid red; /*Criando uma borda pra DIV, 2 pixels e solid porque é uma linha contínua*/
    height: 100vh; /*Está falando pra DIV ocupar 100% da tela*/
    display: flex; /*Posiciona tudo lado a lado*/
    flex-direction:column;  /*Corrigindo posião*/
    align-items: center; /*Centraliza a DIV na parte superior do site*/
    justify-content: center; /*Finaliza o processo de centralizar a DIV no meio do site*/
}

.display {
    background-color: #2C2E43;
    width: 300px; /*Definindo a largura da DIV*/
    height: 200px; /*Definindo a altura da DIV*/
    text-align: center; /*Serve apenas para alinhar o centro superior, não é o caso da DIV Relógio que possui mais propriedades*/
    line-height: 200px; /*Altura da linha, tem que ser igual a altura da DIV para alinhar no centro*/
    border-radius:10px; /*Arredondar os cantos*/
    color: #fff;
    font-size: 7rem;/*Cada REM vale 16 Pixels, estou aumentando o tamanho da fonte*/
    border: 10px solid #ffd523;
    font-family: "Qahiri", sans-serif;
    
}

footer {
    font-size: 2rem;
    margin-top:10px ; /*Distanciando um pouco da DIV de cima*/
    color: #333;
    font-family: "Barlow Condensed", sans-serif;
